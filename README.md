## Welcome to the adventure game "Can you Escape?!"

You have formed an aquaintance with a woman named Kelly. You want to be friends, but Kelly says you must first pass a test to be her friend. She invites you to her house for a cup of tea, but when you enter the house, she locks the door and traps you inside! She says that you must find a way to escape using the clues she left behind. If you figure it out you will have passed the test and can be her friend. 

In order to escape you must find a 4 digit code to unlock the front door. Search each room for clues to help you escape. Each room holds at least one clue and there are two other things that will need to be unlocked before you can unlock the front door. One of those is a room. This room requires a key to unlock it; once you've found the key it will be added to your inventory and the next time you try and enter that room it will automatically let you in. Command options will be given to you, please use those only. When you have figured out the code, first use the unlock door command, and after successfully unlocking it, use a directional command to leave the house. Enjoy the game and good luck!
